# 单宿主机面板范围（已确认）

## 目标形态
- 直接安装在单台宿主机上
- 仅管理员登录
- 仅支持 LXC
- 默认镜像仅两种：alpine/3.20、debian12
- 安装时自动预下载默认镜像

## 保留功能（MVP）
- 自动识别宿主机系统（Linux 发行版）
- 自动识别宿主机架构（amd64/arm64）
- 单管理员认证与登录
- LXC 实例管理：创建、启动、停止、重启、重装、删除、重置密码、终端
- 资源限制：CPU、内存、磁盘、IO、带宽、流量
- 任务队列：waiting/running/success/failed/timeout
- 操作审计日志
- 防火墙默认全开放（不提供规则编辑）

## 明确移除
- 多宿主机管理
- 普通用户体系、消息通知
- 快照功能
- 镜像管理、镜像导入导出（仅保留安装时预置的两个默认镜像）
- 迁移与数据流转全部功能
- 防滥用与安全增强模块
- Prometheus 指标导出
- 备份与恢复
- 版本升级与回滚
- API 与外部集成全部功能
- 兑换码、收费、套餐（此前已确认移除）

## 备注
- 当前安装脚本已支持自动检测 OS 与架构，并在安装阶段预拉取默认镜像。
